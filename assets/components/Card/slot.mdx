## Card 插槽

| 插槽名称           | 说明                       | 参数                      |     |
| ------------------ | -------------------------- | ------------------------- | --- |
| actions            | 卡片操作组，位置在卡片底部 | -                         |     |
| cover              | 卡片封面                   | -                         |     |
| customTab          | 自定义 tabList tab 标签    | { item: tabList[number] } |     |
| extra              | 卡片右上角的操作区域       | -                         |     |
| tabBarExtraContent | tab bar 上额外的元素       | -                         |     |
| title              | 卡片标题                   | -                         |     |
