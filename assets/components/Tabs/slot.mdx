### Tabs 插槽

| 插槽名称     | 说明                            | 参数              |     |
| ------------ | ------------------------------- | ----------------- | --- |
| addIcon      | 自定义添加按钮                  | -                 | -   |
| leftExtra    | tab bar 上左侧额外的元素        | -                 | -   |
| moreIcon     | 自定义折叠 icon                 | -                 | -   |
| renderTabBar | 替换 TabBar，用于二次封装标签头 | { DefaultTabBar } |     |
| rightExtra   | tab bar 上右侧额外的元素        | -                 | -   |
