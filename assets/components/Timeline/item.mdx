## API

| 参数     | 说明                                            | 类型              | 默认值 | 版本 |
| -------- | ----------------------------------------------- | ----------------- | ------ | ---- |
| color    | 指定圆圈颜色 `blue, red, green`，或自定义的色值 | string            | blue   |      |
| dot      | 自定义时间轴点                                  | string\|slot      | -      |      |
| label    | 设置标签                                        | string \| slot    | -      | 3.0  |
| position | 自定义节点位置                                  | `left` \| `right` | -      |      |
