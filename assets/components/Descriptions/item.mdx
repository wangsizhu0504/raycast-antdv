### Item props

| 参数         | 说明           | 类型                    | 默认值 | 版本  |
| ------------ | -------------- | ----------------------- | ------ | ----- |
| contentStyle | 自定义内容样式 | CSSProperties           | -      | 2.2.0 |
| label        | 内容的描述     | string \| VNode \| slot | -      |       |
| labelStyle   | 自定义标签样式 | CSSProperties           | -      | 2.2.0 |
| span         | 包含列的数量   | number                  | 1      |       |

> span 是 Descriptions.Item 的数量。 span={2} 会占用两个 DescriptionsItem 的宽度。
