## API

| 参数             | 说明                              | 类型    | 默认值 |
| ---------------- | --------------------------------- | ------- | ------ |
| autofocus        | 自动获取焦点                      | boolean | false  |
| checked(v-model) | 指定当前是否选中                  | boolean | false  |
| disabled         | 禁用 Radio                        | boolean | false  |
| value            | 根据 value 进行比较，判断是否选中 | any     | -      |
