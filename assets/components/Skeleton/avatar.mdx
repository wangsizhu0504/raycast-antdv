## API

| 属性  | 说明                 | 类型                                      | 默认值 |
| ----- | -------------------- | ----------------------------------------- | ------ |
| shape | 指定头像的形状       | `circle` \| `square`                      | -      |
| size  | 设置头像占位图的大小 | number \| `large` \| `small` \| `default` | -      |
